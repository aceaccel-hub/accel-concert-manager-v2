# 🎼 아첼 연주회 관리 프로그램

## 📋 설치 및 실행 가이드 (윈도우)

### ✅ 1단계: Node.js 설치

프로그램을 실행하려면 먼저 **Node.js**를 설치해야 합니다.

1. [https://nodejs.org/](https://nodejs.org/) 방문
2. **LTS (Long Term Support)** 버전 다운로드 및 설치
3. 설치 후 컴퓨터 재부팅

**설치 확인:**
- 명령 프롬프트(cmd)를 열고 아래 명령어 입력:
```
node --version
npm --version
```

---

### 🚀 2단계: 프로그램 실행

#### 가장 간단한 방법 (권장):

이 폴더에서 **START.bat** 파일을 더블클릭하세요.

![](./images/start-bat.png)

- 자동으로 필요한 패키지가 설치됩니다 (처음 한 번만)
- 브라우저가 자동으로 열립니다
- 프로그램이 **http://localhost:5175** 에서 실행됩니다

#### 명령 프롬프트로 실행:

```bash
npm run preview -- --host 0.0.0.0 --port 5175
```

---

### 💾 데이터 저장

모든 데이터는 **브라우저의 로컬 저장소(IndexedDB)**에 저장됩니다.

- **자동 저장**: 입력한 데이터는 자동으로 저장됩니다
- **로컬 데이터**: 이 컴퓨터에만 저장되며 외부로 전송되지 않습니다
- **백업**: 데이터를 보호하려면 주기적으로 프로그램에서 내보내기 기능을 사용하세요

---

### 🔧 트러블슈팅

#### Q1: START.bat를 클릭해도 아무것도 안 나와요

**A:** 명령 프롬프트가 빠르게 닫혀서 그럴 수 있습니다.

1. 명령 프롬프트(cmd) 열기
2. 이 폴더로 이동: `cd C:\Users\사용자명\Desktop\아첼연주회관리프로그램`
3. 아래 명령어 실행:
```
npm run preview -- --host 0.0.0.0 --port 5175
```

#### Q2: "npm이 인식되지 않습니다"

**A:** Node.js가 설치되지 않았거나 경로 설정이 필요합니다.

1. [https://nodejs.org/](https://nodejs.org/) 에서 LTS 버전 설치
2. 컴퓨터 재부팅
3. 명령 프롬프트를 새로 열어서 다시 시도

#### Q3: 포트 5175가 이미 사용 중입니다

**A:** 다른 포트를 사용하세요:

```
npm run preview -- --host 0.0.0.0 --port 8080
```

그 다음 브라우저에서 `http://localhost:8080` 으로 접속하세요.

#### Q4: 브라우저가 자동으로 안 열려요

**A:** 수동으로 브라우저를 열고 다음 주소 입력:
```
http://localhost:5175
```
